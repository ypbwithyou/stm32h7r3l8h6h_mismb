## 2，量程范围

0 -> gain=0(1倍), pga=1 (2.5V)
1 -> gain=0(1倍), pga=2 (1.25V)
2 -> gain=1(10倍), pga=1 (0.25V)
3 -> gain=1(10倍), pga=2 (0.125V)
4 -> gain=0(10倍), pga=128 (0.01953125V)
5 -> gain=1(10倍), pga=128 (0.001953125V)

根据：ChannelTableElem.nInputRange 需要修改

## 2，桥路类型

0=全桥
1=半桥
2=1/4桥

根据：ChannelTableElem.nVar1 
应力应变：bit15--bit0 --> 桥路类型索引nBridgeType（1-8），bit31--bit16 --> 分流校准类型（0--10）;

桥路使能: 根据：ChannelTableElem.nGroupID

桥路类型定义 (对应 nVar1 bit15-0)
映射规则:
    1-2  -> 1/4桥
    3-4,9-> 1/2桥
    5-7  -> 全桥
    8    -> 配置错误 (默认按1/4桥处理)
